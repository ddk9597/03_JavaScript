함수(function)

        - 목표하는 결과를 만들기 위한 코드 묶음
        - 코드를 {}(블록)으로 묶어 작성하고 붙여진 이름을 호출해서 사용

        * 함수 선언

        * 함수 정의

        * 함수 호출

        [기본 함수 형태]
        function 함수명(){ // 함수 선언
            // 함수 정의
        }

        함수명() ; // 함수 호출

매개 변수 (Parameter), 전달 인자(Arguments)

        * 매개 변수 (Parameter)
         - 함수 선언 시 () 안에 작성되는 변수
          -> 전달 인자로 부터 받은 값을 저장
          -> 함수 정의 부분에서 사용 가능

        * 전달 인자(Arguments)
         - 함수 호출 시 () 안에 작성되는 값
          -> 함수 매개 변수로 전달되는 값

retrun(반환)

        - 현재 수행중인 함수를 종료하고 호출한 곳으로 돌아감
            -> 함수 호출 구조는 Stack 형식으로 되어있음
            -> FILO(First In Last Out, 선입후출)구조

        - 호출한 곳으로 돌아갈 때 return 옆에 작성된 값을 가지고 돌아갈 수 있다

            ex) return 100; // 호출한 곳으로 100을 가지고 돌아감

        - 함수 마지막에 return 구문이 생략되어 있다면
          자동으로 retrun; 구문이 추가된다

익명함수(이름이 없는 함수)

        [기본함수]
        function 함수명([매개변수]){
            // 함수정의
        }

        함수명(); // 이름이 있으니까 함수명으로 호출 가능하다

        [익명 함수]

        ex) const sumFn = function(a,b){
