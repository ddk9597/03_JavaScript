배열 (Array)

02_변수와 자료형 참조 
        - 변수를 묶음으로 다루는 것(자료구조의 기초)
        - 변수를 여러개 설정 가능
        - 자료구조 : 데이터를 효율적으로 배치하는 것
        
        (JS 변수 : 대입되는 값에 따라서 자료형이 변형)

        -> JS 배열 : 자료형 상관 없이 여러 값을 한번에 다룰 수 있다

        - JS 배열의 특징
        1. 자료형에 제한이 없음 (보통 한가지 자료형만 저장한다)
        2. 길이 제한이 없다 (자동으로 배열 길이가 늘었다, 줄었다 한다)

        [JS 배열 선언 방법]

        1. const arr = new Array();
          - 새로운 배열을 만들겠다는 객체지향 언어
          - 비어있는 0칸짜리 배열

        2. const arr = new Array(3);
          - 3칸짜리 배열 생성
          - 각 칸의 값은 비어있음(empty)

        3. const arr = [];
          - 비어있는 0칸짜리 배열을 생성한다(1번과 같은 뜻)
          - [] : 배열 기호

        4. const arr = [ 값1, 값2, 값3 ... , 값n ] ;
          - n칸짜리 배열 생성
          - 각 칸이 값으로 채워져 있음

        3, 4번 가장 많이 씀
