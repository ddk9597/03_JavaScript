객체(object)

        - 객체는 무엇인가
        - 속성과 기능으로 이루어진 것
            - 속성 == 값을 저장하는 변수 (필드)
            - 기능 == 메서드 (객체에 작성된 함수)

        - {} (블록)내에 key : value가 모여있는 형태
            -> Key == 변수명/메서드명 (중복 불가)
            -> value == 값, 함수

        * K : V 가 모여있는 형식 == Map 

        - JS 객체 작성법
        { K:V, K:V, K:V, K:V...}

        [JS객체 생성 방법]
        1. {} : JS 객체 리터럴 표기법을 이용한 생성

        2. 생성자 함수 : 생성자 + new 생성자() 를 이용한 생성

        ** 리터럴 == 값 **
        - value 대신에 사용한다
        - 각 자료형마다 리터럴(값)표기법이 별도로 존재한다
            ex) 문자열(string) -> "", ''
                숫자(number)   -> 1, 2.34, +5, -43
                배열(array)    -> []
                JS객체(object) -> {}
                논리(boolean)  -> true, false

생성자(constructor) 함수

        * 생성자란? (== 생성자 함수)
        - 객체 생성 시 수행되는 기능(함수)
        - 특정한 모양의 객체가 다수 필요한 경우
          이를 만들기 위한 코드의 중복을 최소화 + 재사용성 증가

        * new 연산자 : (메모리에)객체를 새롭게 생성(할당)

        * 생성자 함수를 이용한 객체 생성 방법
         -> const 객체명 = new 생성자함수() ;
