이벤트

        이벤트 : 동작, 행위 ,행사
        -> 브라우저에서 할 수 있는 동작

        ** 이벤트 리스너
        -> 브라우저에서 이벤트가 감지될 때 가지 대기하다가 감지가 되었을 경우
        연결된 기능(함수)를 수행

        -> 이벤틀 이스너는 "on 이벤트명"형식으로 작성

        ** 이벤트 핸들러
        -> 이벤트가 감지되었을 때 수행할 내용을 작성한 함수(function)

인라인 이벤트모델(Inline Event Model)

        종종 사용됨. 많이는 안씀
        HTML요소에 이벤트리스너 = " 이벤트 핸들러"형식을 작성하는 방법
        하나의 요소에 동일한이벤트 핸들러를 중복해서 작성불가
        하나의 요소에 동일한 이벤트핸들러 중복 작성 불가
        왜? 이전에 작성된 이벤트핸들러를 새 이벤트핸들러가 덮어씀
