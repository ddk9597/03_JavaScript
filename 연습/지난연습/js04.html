비교 연산자

        좌변과 우변을 비교해서 맞을 경우 true, 아닐 경우 false를 반환하는 연산자
        연산 결과로 놀리 값(boolean)을 반환

입력된 두 수가 같은지 확인




왼쪽에 입력된 수가 오른쪽에 입력된 수보다 큰지 확인하기


배수가 맞는지 확인하기

결과 값 :

And, Or, 논리 부정 연산

        And ( && )연산 : 및. 두 명제 모두 참이어야지 true를 반환한다
                         false를 반환하는 경우 : T,F/ F,F/ F,T/

        Or ( || ) 연산 : 두 명제 중 하나만 참이어도 참인 명제
                         두 명제 중 둘 모두 거짓이어야지 false를 반환하는 명제

        논리 부정 연산( ! ) : 논리 값을 반대로 바꾸는 연산자
        ex) let test = true;
            !test; // !true - > false
