for 문

        반복하는 제어문
        - 프로그램의 흐름을 제어하는 제어문 중 하나로
          특정 코드를 반복할 때 사용하는 구문
        - if 문과의 차이 : if문은 특정 코드를 건너뛰는 용

        - 끝이 정해져 있는 반복에서 주로 사용
          ex) 몇 번만 반복

        [작성법]

        해석하는 순서
        1) 초기식 
        2) 조건식
        3) 반복 수행할 코드
        4) 증감식

        1~4까지 1회 실행 후 2~4 계속 반복
        조건식이 false가 되면 바로 종료
        
            for(초기식 ; 조건식 ; 증감식){
                조건식이 true인 경우 반복 수행 할 코드
            }

            초기식 : for 문을 제어하는 용도의 변수 선언 
                    (반복 할 만큼 손가락 펴기)

            조건식 : for문의 반복 여부를 지정하는 식 
                    - true일때 반복하고 false일 때 반복 종료
                    - 보통 초기식에 사용된 변수를 이용해서 조건식을 작성한다
                    (반복 한 만큼 손가락 접기)

            증감식 : 초기식에 사용된 값을 변환 시키느 식
                    -> 변환된 초기식에 의해서 조건식의 결과가 달라지게 한다

입력 받은 범위 내 모든 정수의 합

        while문

                - 끝이 정확하게 정해지지 않은 반복에서 주로 사용
          (ex : 어떤 값이 입력 되기 전까지 반복)

          [작성법]

          while(조건식){
            조건식이 t인 경우 반복수행 할 코드 ;

          }
        - 조건식이 t인 경우 내부 코드를 1회 반복 후 다시 조건식 검사 진행
          -> false가 나올 때 까지 무한 반복

        메뉴 주문하기
