변수와 자료형

        변수 : 메모리에 값을 계속 바꿔가며 저장할 수 있는 공강

변수 명명 규칙

- 의미 있는 이름을 작성 - 대 소문자 구분 - 특수문자는 언더스코어만 사용 가능 - 카멜 표기법으로 작성

변수 종류와 특징

var : 값이 변할 수 있음. 중복될 수 있음. 함수레벨 스코프. 많이 안씀 - 변수명이 중복될 수 있음. 중복 선언 가능하며 나중에 선언된 값으로 덮어씌워짐 let : 변수. 블록레벨 스코프 2순위 - 변수명 중복 불가 const : 상수. 블록레벨 스코프 1순위 - 변수명 중복 불가 
        JS는 변수 선언 시 별도의 자료형 지정 안함
        - > 변수에 대입되는 값에 의해서 자료형이 결정됨

        자료형의 종류
        - string 문자열
        - number 숫자
        - boolean 논리 값( true, false만 가능)
        - fuction (함수)
        - undefined(정의되지 않은 변수. 선언만 되고 값이 대입되지 않은 변수)
        - null (참조하는게 없음. 일치하는 요소가 없어서 참조 불가함.)
        - object 객체 : 배열과 자바스크립트 객체가 있음 {k:v, k:v ...}


        객체(object)란 무엇인가
        - 값을 여러개 저장할 수 있는 형태
        - 배열과 자바스크립트 객체가 있음
            - 배열 : 여러 값이 나열되어 있는 것의 묶음
              ex) const numbers = [10, 20, 30]
            - 자바스크립트 객체 : 값을 K:V 형식으로 여러개 저장하는 형태
              ex) const user = {id : "user1", pw : "pass01", userName : "김개동"}
              K(Key. 이름.) : V(value. 값.)

        배열, 자바스크립트 객체에서 각각의 객체의 값 불러오는 방법
        
        1. 배열 
            const numbers = [10, 11, 12]
            - numbers의 첫 번째 값 불러오기 : (numbers[0])
            - numbers의 두 번째 값 불러오기 : (numbers[1])
            - numbers의 세 번째 값 불러오기 : (numbers[2])
        
        2. 자바스크립트 객체
            const user ={id:"머시기", pw:"pass01", userName : {name01}}

            - 객체에 존재하는 값 하나씩 불러오는 방법 - 이름['key'] ;
            console.log(user['id'], user['pw'], user[name])

            - 이름으로 불러오는 방법                  - 이름.key ;
            console.log(user.id, user.pw, user.userName)
