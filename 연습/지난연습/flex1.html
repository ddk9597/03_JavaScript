displat : flex (플렉스 박스)

        요소의 정렬되는 방향, 순서, 요소 간의 간격을 하나 하나 수치적으로 처리하지 않아도
        알아서 유연하게 처리하는 방식

         flex 이용 시 반드시 알아야 할 것

         1. flex의 구성
        - flex container : 정렬이 필요한 요소(item)를 감싸는 요소
        - item : 정렬이 적용될 요소

        (fleax container용 속성, item용 속성이 구분되어 있음)

        2. flex의 축(axis)
        - 중심축 (main axis)
        - 교차축, 반대축 (cross axis)
