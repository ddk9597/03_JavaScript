조건문

if 문

        
            if(조건식){
                조건식이 treu일 경우 실행할 코드 ;
            }
        

        * 조건식 : 결과가 true, false 인 식 (연산)
        * 조건문 : 조건식의 결과에 따라 실행을 제어하는 구문 (if, switch문)

입력받은 수가 양수인지 아닌지 검사

if-else 구문

        
            if(조건식) {
                조건식이 true일 경우 수행 할 코드 ;
            }

            else{ 
                조건식이 false일 경우 수행할 코드 ;
            }

0~100 난수를 발생시켜 홀,짝 판별하기

if else 구문을 이용하여 어린이 청소년 성인 구분하기

        나이를 입력 받는다.
        0~13 : 어린이
        14~ 19 : 청소년
        19~150 : 성인
        0 미만 또는 150 초과 : 값을 잘못 입력하셨습니다.
        미입력 : 값을 입력하세요.
