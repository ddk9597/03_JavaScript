Script 언어란? 

        한 줄 읽고 바로 실행하는 언어
        기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도의 언어
        별도의 소스코드 컴파일을 하지 않고(컴파일 : 2진수 번역)
        interpreter(인터프리터, 한 줄씩 코드를 읽어서 해석하는 기능)를 이용해서
        코드를 바로 실행한다

        -장점 : 컴파일(2진수 번역) 에 소요되는 시간이 없음
        -단점 : 프로그램 실행 시간이 상대적으로 느리다

JavaScript란?

        웹 브라우저(크롬, 엣지, 웨일, 파이어폭스 등)에서
        주로 사용하는 interpreter방식의 객체지향 프로그래밍 언어

        ECMA Script 표준을 따라하는 대표적인 웹 기술
        (JS버전 이름 : ES5, ES6, ES7 ...) ES6 기준으로 공부한다

Java Script 코드 작성 방법

  1. 브라우저 개발자도구 -> Console 에 직접 작성

  2. html내부에 script 태그를 이용해서 작성(Internal 방식)


  3. html 외부에 존재하는 .js 파일을 연결해서 사용하는 방법(External), 제일 많이 쓰는 방식


  4. 태그에 직접 JS 코드를 작성(inline방식, 권장안함)
